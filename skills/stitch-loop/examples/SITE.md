# Site Document — DevTracker

## 1. Vision

DevTracker is a developer productivity dashboard that tracks coding time, project progress, and team activity. Clean, modern UI with Symbiote.js components, built as a SPA with path-based routing.

## 2. Stitch Project

- **Project ID**: `4044680601076201931`
- **Project Name**: DevTracker Dashboard

## 3. Technology

- Symbiote.js 3.x — component framework
- AppRouter — path-based SPA routing
- Native CSS — custom properties from DESIGN.md
- ESM — importmap in index.html

## 4. Sitemap

- [x] `/` — Home (Dashboard overview with key metrics)
- [x] `/projects` — Projects (List of tracked projects with stats)
- [ ] `/activity` — Activity (Timeline of recent commits and events)
- [ ] `/settings` — Settings (User preferences and integrations)
