# UI/UX Keywords for Stitch Prompts

Use these keywords to add precision to your Stitch prompts. Organized by category.

## Layout Keywords

| Keyword | Effect |
|---------|--------|
| card grid | Multi-column grid of equal-height containers |
| masonry layout | Pinterest-style variable-height grid |
| sidebar layout | Fixed side panel with scrollable main content |
| split screen | 50/50 two-panel layout |
| hero section | Full-width prominent top section with CTA |
| sticky header | Navigation that stays fixed on scroll |
| vertical feed | Scrollable timeline-style list |
| dashboard layout | Multi-widget grid with metrics and charts |

## Component Keywords

| Keyword | Effect |
|---------|--------|
| navigation bar | Horizontal top navigation with links |
| breadcrumbs | Path-based navigation trail |
| tab bar | Horizontal tabs for content switching |
| accordion | Expandable/collapsible sections |
| modal dialog | Centered overlay with backdrop |
| dropdown menu | Expandable menu triggered by click |
| tooltip | Hover-triggered info popup |
| toast notification | Temporary message at screen edge |
| search bar | Input with icon and clear button |
| date picker | Calendar-based date selector |
| progress bar | Horizontal loading/completion indicator |
| avatar | Circular user image with fallback |
| badge | Small label/count indicator |
| chip/tag | Removable label element |
| toggle switch | Boolean on/off control |
| skeleton loader | Animated placeholder during loading |

## Visual Style Keywords

| Keyword | Effect |
|---------|--------|
| glassmorphism | Frosted glass effect with backdrop blur |
| neumorphism | Soft embedded/raised appearance |
| flat design | No shadows, solid colors |
| material design | Subtle shadows, defined elevation |
| gradient mesh | Colorful gradient backgrounds |
| duotone | Two-color image treatment |
| high contrast | Strong color differentiation |
| muted tones | Desaturated, calm palette |
| generous whitespace | Open, airy layout |
| dense layout | Compact, information-rich |

## Interactive State Keywords

| Keyword | Effect |
|---------|--------|
| hover state | Visual change on mouse hover |
| active/pressed | Visual change on click |
| focus ring | Visible outline on keyboard focus |
| disabled state | Grayed out, non-interactive |
| loading state | Spinner or skeleton while fetching |
| empty state | Friendly message when no data |
| error state | Red border/text for invalid input |
| selected/active | Highlighted current item |
| expanded/collapsed | Toggled open/closed sections |

## Animation Keywords

| Keyword | Effect |
|---------|--------|
| fade in | Opacity transition on appear |
| slide up | Translate + opacity from below |
| scale in | Grow from small to full size |
| stagger | Sequential animation of list items |
| parallax | Depth effect on scroll |
| smooth transition | 250ms ease timing |
| micro-animation | Subtle feedback on interaction |

## Symbiote-Specific Tag Name Suggestions

When describing components, include suggested custom element names:

| UI Pattern | Suggested Tag |
|-----------|---------------|
| Main navigation | `app-nav` |
| Footer | `app-footer` |
| Generic card | `ui-card` |
| Button | `ui-button` |
| Input field | `ui-input` |
| Modal/Dialog | `ui-modal` |
| Dropdown | `ui-dropdown` |
| Status badge | `status-badge` |
| User avatar | `user-avatar` |
| Data table | `data-table` |
| Page header | `page-header` |
| Metric card | `metric-card` |
| Feed item | `feed-item` |
| Empty state | `empty-state` |
